# AI Tutor - AI-Powered Personalized Learning Platform

**Purpose:** An intelligent learning platform that helps users create personalized courses on any topic using Google's Gemini AI, with interactive features for enhanced learning experiences.

## Key Points

1. **AI-Powered Course Generation**: The platform leverages Google Gemini AI (2.0 Flash model) to automatically generate comprehensive course content with HTML formatting, interactive quizzes, and knowledge tests on any topic the user desires.

2. **Full-Featured Learning Experience**: Includes Google Authentication, progress tracking, bookmarks, notes, a Teaching Assistant powered by AI, and dark/light mode support - all built with Next.js 15, React 19, Prisma ORM, and PostgreSQL.

3. **Modern & Scalable Architecture**: Built with a unified Next.js backend architecture, easily deployable on Vercel, VPS, or Docker, with a complete REST API for courses, user progress, notes, bookmarks, and subscription management.
